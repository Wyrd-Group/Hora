  Purpose: Sports value betting detection
  Algorithm:
    - Fetch live odds & team records (ESPN API stub)
    - Calculate fair win prob: W² / (W² + L²)
    - Compare model prob vs implied prob from odds
    - Size bets with Kelly criterion
    - Alert when edge > min_edge_pct
  Status: Complete, v1 with Pythagorean expectation

DEPLOYMENT

QUICK START:
  python -m sentinel.daemon --test          # Verify it works
  python -m sentinel.daemon --once          # Run all scanners once
  python -m sentinel.daemon                 # Run daemon continuously

CONFIGURATION:
  Edit sentinel/config.py to adjust:
    - active_hours (when to send alerts)
    - bankroll_eur (total capital)
    - min_edge_pct (minimum edge to alert)
    - max_kelly_pct (max bet size per alert)
    - stock_watchlist, crypto_watchlist
    - alert_channel ('console', 'whatsapp', 'telegram')

MONITOR:
  tail -f sentinel/alerts_log.json          # Watch alerts as they arrive
  python -m sentinel.daemon > sentinel.log  # Log to file
