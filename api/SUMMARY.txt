KEY FEATURES
================================================================================

RESPONSE FORMAT:
All endpoints return consistent JSON wrapper:
  {
    "status": "ok" | "error",
    "data": {...},
    "engine": "engine_name",
    "timestamp": "ISO-8601",
    "message": "error message if applicable"
  }

ERROR HANDLING:
- HTTP exceptions return proper status codes
- Unhandled exceptions caught and logged
- Stack traces available in development mode
- Detailed error messages in responses

LOGGING:
- Timestamp + module + level + message
- Configurable via LOG_LEVEL environment variable
- Separate console output for INFO, ERROR, DEBUG
- Request logging middleware

CORS:
- Enabled for localhost (all ports)
- Easily configurable for production domains
- Handles preflight OPTIONS requests

SECURITY:
- Input validation via Pydantic models
- CORS configuration for localhost
- Environment-based configuration
- Ready for HTTPS (reverse proxy)

PERFORMANCE:
- Async/await for concurrent requests
- Uvicorn ASGI server (production-grade)
- Compatible with Gunicorn + Uvicorn
- Docker containerizable
