## Troubleshooting

### SVG Not Displaying
- Ensure viewBox="0 0 800 600" on the SVG element
- Check that stroke width is appropriate for your zoom level
- Verify path data is not truncated

### Performance Issues
- Lazy load circuit data if not using all circuits
- Consider debouncing circuit selection changes
- Cache rendered SVGs if possible
