Architecture & Systems

The platform uses a client-heavy architecture where all game simulation, price engines, and AI predictions run in the browser. This eliminates per-user compute costs and enables offline play. The server layer handles only authentication, data persistence, and multiplayer coordination.

📊 Market Simulation Engine

  • Geometric Brownian Motion (GBM) with jump-diffusion
  • Monte Carlo prediction engine (2,000 simulation paths)
  • On-demand price extension for infinite live markets
  • Campaign time system (1 real sec = 5 in-game sec)
  • Real-time OHLC candlestick chart with indicators
  • Multi-asset support: equities, crypto, forex, commodities

🌍 Living World Engine

  • Interactive 3D globe (deck.gl + MapLibre)
  • Geopolitical event generation (30 event types)
  • Event-to-market price impact pipeline
  • Country-level economic indicators
  • Dynamic trending events system
  • Node acquisition and empire expansion

🎓 ECFL Education System

  • 3-band certification standard (F1/F2/F3)
  • Structured curriculum with progressive modules
  • 30-question exam banks per band
  • Athena AI tutor (context-aware assistance)
  • CEO XP leveling system with badges
  • Certificate generation on exam completion

🏢 Empire Management

  • 12-tab business management dashboard
  • Division management with P&L tracking
  • Equity panel with shareholder simulation
  • Transport network with vehicle fleet
  • Intelligence hub with market intelligence
  • Bureaucracy system with compliance fines

⚔️ Multiplayer Arena

  • Match HUD with live P&L leaderboard
  • Campaign mode with persistent world state
  • Spectator mode for ongoing matches
  • Cross-player trade competition scoring

💎 Economy & Monetization

  • Battle Pass (free + premium tracks)
  • 4-tier subscription model (Recruit → Director)
  • AP (Aegis Points) microtransaction currency

🏛️ Political Simulation

  • Rank-based progression (Citizen → President)

🤖 AI Integration

Infrastructure & Cost Model

The client-heavy architecture means the platform can scale to tens of thousands of concurrent users with minimal server costs. All price simulation, AI predictions, game logic, and rendering execute in the user's browser.

Service Tier Monthly Cost What It Handles
Netlify (Hosting) Free / Pro $0 – $19/mo Static CDN, SSL, serverless functions, global edge
Supabase (Backend) Pro $25/mo Auth, 8GB PostgreSQL, Realtime channels, Edge Functions
Compute (Simulation) Client-side $0 All game logic, prices, AI predictions run in browser
Total $25 – $44/mo Supports thousands of DAU with zero per-user compute cost

Per-user marginal cost approaches $0 because all heavy computation (price ticks, Monte Carlo simulations, chart rendering, 3D globe) executes client-side. Server handles only lightweight auth tokens and Realtime channel messages.
